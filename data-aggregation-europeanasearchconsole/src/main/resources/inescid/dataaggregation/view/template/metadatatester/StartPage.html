About the prototype

This prototype is a work in progress. It aims to support data providers of Europeana in checking the compliance of their implementation of solutions for delivering data to Europeana.

This console supports checking of two types of resources:

  • Schema.org included in a webpage (as used for SEO)
  • IIIF manifest with structured metadata in Schema.org or EDM (seeAlso references)

For using the console, follow the link for the type of solution to be checked, and use URL addresses of webpages or of IIIF manifests.
